# Коммуналка и копилка (GitHub Pages)

Простой мобильный сайт (одна страница) для показа:
- коммуналки по кв.1
- коммуналки по кв.2
- итога коммуналки
- остатка из 5000 ₴
- накопления “копилки” (сумма остатков по месяцам)

Сайт сделан под старый смартфон (примерно 5" экран): большие цифры, высокий контраст, без фреймворков.

---

## 1) Как включить GitHub Pages

1. Создай репозиторий на GitHub, например: `rent-ledger`
2. Залей в него файлы:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `data.json`
   - `README.md`
3. Открой в репозитории:
   - **Settings → Pages**
4. В разделе **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. Сохрани. GitHub даст ссылку на сайт (обычно вида `https://<user>.github.io/<repo>/`)

---
